# WASM

## Command Lines

- `emcc main.cpp -std=c++17 -lSDL2 -O3 -s WASM=1 -O3 -o hello.html --shell-file template.html --preload-file ./resources`
- `python3 -m http.server`
- `pkg-config --libs sdl2`
- `source ./emsdk_env.sh`
